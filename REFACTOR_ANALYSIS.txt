2.1 CRITICAL: CONFLICTING PostCSS CONFIGURATIONS
──────────────────────────────────────────────────

Two files exist with different Tailwind versions:

File 1: postcss.config.js (ACTIVE - Tailwind v3)
  Config:
    module.exports = {
      plugins: {
        tailwindcss: {},
        autoprefixer: {},
      },
    };

File 2: postcss.config.mjs (CONFLICTING - Tailwind v4)
  Config:
    export default {
      plugins: {
        '@tailwindcss/postcss': {},
      },
    };

Problem: Build may pick one over the other, causing undefined behavior.

Recommendation:
  IF using Tailwind v3: DELETE postcss.config.mjs
  IF upgrading to v4: DELETE postcss.config.js and update package.json dependencies

Action Required: Consolidate to ONE file immediately.
